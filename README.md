# 📖 Présentation

Ce repository documente mon apprentissage de **React** dans le cadre du cursus proposé par freeCodeCamp.

Chaque dossier correspond à un exercice ou un mini projet permettant de pratiquer les concepts suivants :

- composants React
- gestion du state
- props
- gestion des événements
- manipulation dynamique du DOM
- organisation d'une application front-end moderne


---

# 🎯 Objectifs pédagogiques

Ce projet a pour but de :

- Comprendre le fonctionnement des **composants React**
- Apprendre à gérer le **state et les props**
- Manipuler les **événements utilisateurs**
- Construire des **interfaces interactives**
- Structurer une application React de manière claire
